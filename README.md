# CS
CS 공부 내용 정리


# 1일 1 CS 질문 정리
2025.11.26(수) - 운영체제(OS)의 정의와 핵심적인 역할(자원 관리, 프로세스 및 스레드 관리, 사용자 인터페이스 제공)에 대해 공부함.

2025.11.27(목) - 프로세스와 스레드의 정의와 차이점을 메모리 관점과 오버헤드 관점에서 구분하여 공부함. 멀티스레드와 멀티프로세스 환경에 대해서도 공부함.

2025.11.28(금) - CPU 스케줄링에 대해 학습함. 선점형과 비선점형 스케줄링 방식의 차이와 특징에 대해 학습함.

2025.11.29(토) - 가상메모리(Virtual Memory)의 정의와 사용 목적(주소 공간 확장, 다중 프로그래밍 및 자원 보호)에 대해 공부함. 페이징의 원리, 외부/내부 단편화 문제의 차이에 대해 학습함.

2025.11.30(일) - 프로세스 동기화와 동기화 문제를 야기하는 핵심 원인인 경쟁조건(Race Condition)에 대해 학습함. 동기화 도구인 뮤텍스(Mutex)와 세마포어(Semaphore)의 차이에 대해 학습함.

2025.12.01(월) - 트랜잭션(Transaction)이 무엇인지, ACID에 대해 학습함. 

2025.12.02(화) - 데이터베이스 인덱스의 정의와 사용 목적(검색 속도 향상, 정렬 및 그룹화 작업 속도 향상)에 대해 학습. 또한, B-tree의 특징에 대해 학습함.

2025.12.03(수) - OSI 7계층 모델의 각 계층을 정의함. 하위 3계층을 구분하는 기준(물리적 연결 및 경로 문제) 및 상위 3계층(애플리케이션 및 데이터 처리 문제)에 대해 학습함.

2025.12.04(목) - 객체지향프로그래밍의 정의와 OOP의 4대 핵심 특성인 캡슐화와 상속, 추상화, 다형성에 대해 학습함.

2025.12.24(수) - 객체지향 설계의 5대 원칙인 SOLID가 각각 무엇을 의미하는지에 대해 학습함. 그리고 테스트 자동화 코드를 작성할 때 SOLID 원칙을 지키면 어떤 이점을 얻는지 각 원칙 별로 정리하여 학습했음.

2025.12.26(금) - 배열과 연결리스트의 물리적인 메모리 저장 방식의 차이에 대해 학습함. 그리고 이 차이로 인해 발생하는 데이터 조회와 삽입/삭제 시의 시간 복잡도 차이에 대해 학습함.

2025.12.29(월) - 이진탐색트리(BST)의 정의와 데이터를 배치하는 핵심 규칙에 대해 학습함. 그리고 이상적인 경우와 Worst Case의 경우 각각 시간 복잡도가 어떻게 되는지 비교하여 학습했고, 만약 편향 트리 형태가 된다면 어떤 문제가 발생하는지(연결리스트와 다를 바 없어져 시간 복잡도가 (O(n))까지도 탐색 속도가 느려질 수 있음)에 대해 학습하였음.

2025.12.30(화) - 버블정렬과 퀵정렬의 차이와 동작원리에 대해 학습함. 버블정렬(O(n^2))과 퀵정렬(O(n log n))의 시간 복잡도에 대해 학습하고, 데이터가 거의 정렬되어 있는 상태에서 퀵 정렬을 사용할 때 발생할 수 있는 문제점(트리가 한쪽으로 치우치게 되어 성능이 급격하게 저하되는 현상)에 대해 학습하였음. 이를 방지하기 위해서는 중간값 피벗 방식이나 랜덤 피벗을 사용함.

2025.12.31(수) - 해시테이블과 해시 함수, 해시 충돌의 정의와 역할에 대해 학습함. 또한, 해시충돌을 해결하기 위한 개별 체이닝과 오픈 어드레싱의 차이에 대해 학습함.
